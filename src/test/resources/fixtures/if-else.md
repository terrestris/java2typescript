# If and else

## if
options: methodBody
```java
if (3 == 5) {
    return "equal";
}
```
```typescript
if (3 === 5) {
    return "equal";
}
```

## if else
options: methodBody
```java
if (3 <= 5) {
    return "less than or equal";
} else {
    return "greater";
}
```
```typescript
if (3 <= 5) {
    return "less than or equal";
}
else {
    return "greater";
}
```

## if else if else
options: methodBody
```java
if (3 > 5) {
    return "greater";
} else if (3 < 5) {
    return "smaller";
} else {
    return "equal";
}
```
```typescript
if (3 > 5) {
    return "greater";
}
else if (3 < 5) {
    return "smaller";
}
else {
    return "equal";
}
```
